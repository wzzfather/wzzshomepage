nano 常用快捷键学习

结论先行：掌握以下快捷键，可在 Linux 下熟练使用 nano 编辑器修改、复制、粘贴、查找替换及跨页操作。

1. 文件操作

快捷键效果说明
Ctrl+O保存文件提示确认文件名，回车即可覆盖
Ctrl+X退出 nano若未保存，会提示是否保存
Ctrl+R插入文件将另一个文件内容插入到当前光标位置
Ctrl+J对齐段落整理当前段落的缩进与换行

2. 光标移动

快捷键效果说明
Ctrl+A移动到行首快速跳到当前行开头
Ctrl+E移动到行尾快速跳到当前行结尾
Ctrl+Y向上翻页类似 PageUp
Ctrl+V向下翻页类似 PageDown
Ctrl+_跳到指定行/列按行号快速定位光标

3. 文本选取、剪切、复制、粘贴

快捷键效果说明
Ctrl+^开始标记文本设置标记起点，用于复制或剪切
Ctrl+K剪切当前行/选中内容剪切到 nano 内部剪贴板
Alt+6复制选中内容与 Ctrl+K 区别：复制不删除原文
Ctrl+U粘贴从 nano 内部剪贴板粘贴到光标位置
Ctrl+^ + 翻页跨页选择文本标记起点后可使用方向键/PageUp/PageDown 选择大段文本

4. 查找与替换

快捷键效果说明
Ctrl+W查找文本输入关键字跳转匹配位置
Ctrl+\\替换文本可逐个确认或一键替换所有
Alt+W查找下一个匹配重复搜索当前关键字

5. 撤销与重做

快捷键效果说明
Alt+U撤销撤销上一次操作
Alt+E重做恢复撤销操作

6. 文件浏览与其他实用操作

快捷键效果说明
Ctrl+T拼写检查启动拼写检查程序（若系统有安装 aspell）
Ctrl+C显示光标位置行号与列号信息
Ctrl+G显示帮助列出 nano 所有快捷键

7. 跨页选中文本示例操作

  1. 光标移动到选择起点，按 Ctrl+^。
  2. 使用方向键或 PageDown/Ctrl+V 移动光标跨页，扩展选择。
  3. 选择完成后按 Alt+6 复制或 Ctrl+K剪切。
  4. 移动光标到目标位置，按 Ctrl+U 粘贴。
记忆口诀：Ctrl+^ → 翻页选 → Alt+6/ Ctrl+K → Ctrl+U。

8. 小结一下

  • Ctrl+O 保存 ✅
  • Ctrl+X 退出 ✅
  • Ctrl+K / Alt+6 + Ctrl+U 剪切/复制粘贴 ✅
  • Ctrl+^ + 翻页可跨页选中 ✅
  • Ctrl+W / Ctrl+\\ 查找与替换 ✅
  • Alt+U / Alt+E 撤销/重做 ✅
